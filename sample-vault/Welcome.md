# Welcome to WebObsidian 🔮

This is a **self-hosted, Obsidian-compatible** web app. Your vault is just a folder of
Markdown files on the server.

## Try these

- Open the [[Notes/Ideas]] note via the wikilink.
- Search for `tag:welcome` in the search panel.
- Toggle Reading view from the toolbar.

> [!tip] Tip
> Press `Cmd/Ctrl + P` to open the command palette.

## Features

- [x] Markdown editor (CodeMirror 6)
- [x] QMD full-text search
- [x] GitHub sync with Git LFS
- [x] Agent API with API keys
